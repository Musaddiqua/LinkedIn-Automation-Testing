# LinkedIn-Automation-Testing
Selenium automation for LinkedIn login


## Objective
This project automates LinkedIn login functionality and validates the home feed using Selenium WebDriver.

## Tech Stack
- Python
- Selenium WebDriver

## Features
- Automated login with valid credentials
- Verification of successful login
- Navigation to home dashboard
- Extraction of top 5 posts from feed
- Exception handling and screenshot capture on failure

## Project Structure
- linkedin_automation.py  -> Main automation script

## How to Run
1. Install dependencies:
   pip install selenium

2. Run the script:
   python linkedin_automation.py

## Notes
- Use valid test credentials
- ChromeDriver must be installed and configured
